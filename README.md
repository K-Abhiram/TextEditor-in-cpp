# TextEditor-in-cpp

A simple terminal-based text editor built in C++ that supports basic file manipulation operations. This project is being developed incrementally to explore file handling, STL containers, and command-line application development.

## Features

- Create/Open text files
- Display file contents with line numbers
- Append multiple lines to a file
- Overwrite the entire file
- Delete a specific line
- Edit an existing line
- Insert a new line at any position
- Menu-driven user interface

## Technologies Used

- C++
- Standard Template Library (STL)
- File Handling (`fstream`)
- `vector`
- `string`
- `getline()`

## Usage

1. Compile the program.
2. Run the executable.
3. Enter the filename you want to work with.
4. Select an operation from the menu.
5. When entering multiple lines, type `/end` on a new line to finish.
